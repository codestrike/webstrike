What is competitive programming ?

Competitive programming is a mind sport, involving participants trying to program according to provided specifications. Competitive programming is recognized and supported by several multinational software and Internet companies, such as Google, Facebook and IBM.

What will be covered in workshop ?

We will cover basics of competitive programming. Also give introductory examples of dynamic programming.

We will also solve competitive programming problems during workshop to demonstrate live working of different solutions of same problem.

What do you need ?

  • Prior programming experience. You should be familiar with atleast one programming language. Note that this workshop is language independent
  • One notebook and pen.
  • Laptop (optional). If you don't have laptop, we will provide you PC.
  • Softwares like Compilers, IDE etc which you will use. We already have softwares required for Python.
